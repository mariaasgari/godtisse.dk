# godtisse.dk

Hjemmeside om forebyggelse af gentagne blærebetændelser og urinvejsinfektioner.

## Teknologier

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Kom i gang

```sh
# Installer dependencies
npm install

# Start udviklingsserver
npm run dev

# Byg til produktion
npm run build
```
